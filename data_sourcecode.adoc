
== 研究に有用なプログラミング言語

.紹介する言語
* R
* Python
* JavaScript

.共通の特徴
* 実行環境の構築が比較的容易
* OSによらず同じコードを使い回せる
* コンパイル無しで実行可能
* 「型」の制約が少ない
* Excelシートやcsv, jsonといった様々なデータ形式を扱える


利点::
  ** 試行錯誤のスピードを早くできる
  ** アルゴリズムに集中できる
欠点::
  ** 処理速度で不利 
  ** データ形式に起因する思わぬバグの可能性

[TIP]
====
* 計算負荷の高い部分をC言語やFortranに委譲することも可能
* 型をつけられる方言もある
====
